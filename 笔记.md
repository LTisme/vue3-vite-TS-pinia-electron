# 笔记

- public 这个文件夹存放静态资源，但是不会被vite编译进去。
- src
    assets 这个文件夹也是存放静态资源，但会被编译进去。
    components 这个文件夹存放《组件的》
- App.vue 是vue的全局入口文件
- main.ts 是全局的TS文件，全局的样式、全局的API都可以在这里引入
- index.html 是vite的入口文件，使用vite是这样的，使用webpack则是使用index.js
- package.json 是一些命令和一些依赖
- tsconfig.json 是TS的一些配置
- vite.config.json 是vite的一些配置，比如里面放了一些plugins，使其支持vue的解析
